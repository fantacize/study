# Philosophy Study Studio

This is now a Next.js study app.

What is included:

- Flashcards built from Chapters 1 and 2
- Multiple-choice drill mode
- Free-response prompts with self-scoring checklists
- Matching sets for terms and definitions
- Cram sheet with comparisons and memory anchors

Run it locally:

```bash
cd /Users/fanta/Documents/study
npm install
npm run dev
```

Then visit `http://localhost:3000`.

For a production check:

```bash
cd /Users/fanta/Documents/study
npm run build
```
